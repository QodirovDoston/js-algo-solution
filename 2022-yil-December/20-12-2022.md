## [65 Find Out the Leap Year](https://edabit.com/challenge/xKKwvL2zYC8pEvMLG)
**Answer**
```js
function leapYear(year) {
	if ((year % 4 == 0) && (year % 100 != 0) || (year % 400 == 0)) {
		return true
	}
	else {
		return false
	}
}
```
