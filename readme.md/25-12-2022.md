## [Triangle and Parallelogram Area Finder](https://edabit.com/challenge/Z5nLWN9XscsuRi2oT)
**Answer**
```js
function areaShape(base, height, shape) {
	if(shape == "parallelogram"){
		return base * height
	}else if(shape == "triangle"){
		return (base * height) * 0.5
	}
	else {
		return "xato"
	}
}
```
