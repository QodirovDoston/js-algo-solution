### [110. Which Function Returns the Larger Number?](https://edabit.com/challenge/o7TwicAHWuMkjbDqQ)
**answer**
```js 
function whichIsLarger(f, g) {
	let fun1 = f()
	let fun2 = g()
	if(fun1 > fun2){
		return "f"
	}else if(fun1 < fun2){
		return "g"
	}else{
		return 'neither'
	}
}
whichIsLarger(() => 5, () => 10)
```
